このフォルダにFigmaからエクスポートした以下の画像を配置してください。

【必要な画像 (PNG)】
1. case01.png  — CASE 01のイラスト (110×182px)
2. case02.png  — CASE 02のイラスト (112×180px)
3. case03.png  — CASE 03のイラスト (163×164px)
4. wave-mask.png — Hero背景の波形シェイプ (1378×672px、白黒マスク用PNG)

【取得方法】
Figma上で各イラスト/波形シェイプを選択し、右パネルの「Export」から
PNG (1x または 2x) でエクスポートして、このフォルダに保存してください。

なお、波形は「マスク用」として使われます。シェイプを白で塗りつぶした
PNG(背景透明)としてエクスポートすると、CSSでグラデーションを
重ねて再現できます。
